Для создания справочных текстов существует 3 атрибута: android:help, android:openTextFile и android:longOpenTextFile. Атрибут android:help Значение — любой ваш собственный текст(см. Ниже). Атрибут android:help создаёт окно помощи, в котором отображается текст, указанный вами в его значении. Для того, чтобы это окно помощи запустить, необходимо использовать код -514.
Примеры: <Key android:codes="-514" android:help="Текст1" /> При коротком нажатии на клавишу откроется окно помощи со справочным текстом Текст1. <Key android:keyLabel="а" android:longCode="-514" android:help="Текст2" /> При коротком нажатии на клавишу введётся буква а, а при долгом нажатии откроется окно помощи со справочным текстом Текст2. Атрибуты android:openTextFile и android:longOpenTextFile Значение — ссылка на файл txt. Атрибут android:openTextFile открывает указанный вами текстовый файл во встроенном редакторе текста по короткому нажатию. Атрибут android:longOpenTextFile делает то же самое, но по долгому нажатию. 1 способ ⒈ Справочный текст находится в самом коде раскладки. ⒉ Нельзя использовать символы &, <, >, ' и "(см. @wikiarticle_2818). ⒊ Перевод на следующую строку – {55FFFF \n}. ⒋ Большой текст справки сильно замедляет раскладку. ⒌ Нельзя создать содержание. 2 способ ⒈ Справочный текст находится в отдельном txt файле. ⒉ Можно использовать символы &, <, >, ' и ", т.к. это не xml. ⒊ Перевод на следующую строку с помощью Enter. ⒋ Можно размещать большие тексты справки. ⒌ Можно создать содержание. Для этого в начало строки вставьте следующий код: $I: после чего напишите название заголовка на этой строке. Чтобы открыть содержание, нажмите ≡ в правом верхнем углу окна справки.
