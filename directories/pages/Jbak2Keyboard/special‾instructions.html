• $userMenu — позволяет создать своё собственное . $userMenu @TITLE:… @COLUMN:… … Основное содержимое специнструкции $userMenu — это кнопки, которые указываются после @TITLE:… и @COLUMN:…. Наличие @TITLE: и @COLUMN: необязательно, и их отсутствие равносильно отсутствию текста после них.

    После @TITLE: вы можете написать название меню.

    После @COLUMN: (задаёт число столбцов кнопок меню) вы можете указать
    • 0/0,/0,0 (равносильно отсутствию) — значения и для вертикальной, и для горизонтальной ориентации телефона будут браться из настроек (jbak2 Keyboard → Внешний вид → (Меню) → В портретном режиме/в ландшафтном режиме)
    • x/x,/x,0 — x будет числом столбцов кнопок меню (как для вертикальной, так и для горизонтальной ориентации телефона)
    • 0,y — y будет числом столбцов кнопок меню при горизонтальной ориентации телефона; число же столбцов кнопок меню для вертикальной ориентации телефона будет браться из настроек
    • x,y — x будет числом столбцов кнопок меню при вертикальной ориентации телефона, а y будет числом столбцов кнопок меню при горизонтальной ориентации телефона
