• $userMenu — позволяет создать своё собственное $userMenu @TITLE:… @COLUMN:… … Наличие @TITLE:… и @COLUMN:… необязательно.

    После @TITLE: вы можете написать название меню. После @COLUMN: (задаёт число столбцов кнопок меню) вы можете указать

Если не указывать вообще, указать первое значение {55FFFF 0}, а второе не указывать или оба значения указать {55FFFF 0}, значения и для вертикальной и для горизонтальной ориентации телефона будут браться из настроек (jbak2 Keyboard → Внешний вид → (Меню) → В портретном режиме/в ландшафтном режиме). Если указать в качестве первого значения натуральное число, а второе значение не указывать или указать {55FFFF 0}, указанное в первом значении натуральное число и будет числом столбцов кнопок меню (как для вертикальной, так и для горизонтальной ориентации телефона). Если в качестве первого значения указать {55FFFF 0}, а в качестве второго натуральное число, указанное во втором значении натуральное число будет числом столбцов кнопок меню при горизонтальной ориентации телефона; число же столбцов кнопок меню для вертикальной ориентации телефона будет браться из настроек. Если же в качестве обеих значений указать натуральные числа, то первое значение будет числом столбцов кнопок меню при вертикальной ориентации телефона, а второе значение будет числом столбцов кнопок меню при горизонтальной ориентации телефона. $[0,t]
