По префиксу в названии xml файла Языковой Возможные префиксы: ru, en и т.д.(двухбуквенные коды языков). Коды всех языков мира http://ru.wikipedia.org/wiki/Коды_языков В настройках клавиатуры вы можете включать и отключать различные языки и выбирать для каждого языка разные раскладки. Последняя языковая раскладка открывается: ⒈ Если закрыть и снова открыть клавиатуру, ⒉ При нажатии клавиши с [кодом]@wikiarticle_2822 -2 на любой не языковой раскладке. [Коды]@wikiarticle_2822 переключения языков: -20, -21, -22, -23 и -24 Также можно открыть раскладку определённого языка с помощью значений $$ru, $$en и т.д.(см. @wikiarticle_2820). Открытая таким образом раскладка – временная: кнопка с кодом -2 на открытой таким образом раскладке будет возвращать к последней языковой раскладке. Неязыковой Возможные префиксы: symbol, symbol2, edittext, smile, num и calc. В настройках клавиатуры вы не можете отключить эти виды раскладок, но вы можете выбирать разные раскладки для каждого вида. • {55FFFF symbol} – символьная раскладка. Открывается: ⒈ По нажатию клавиши с кодом -2 на языковой раскладке, ⒉ По нажатию клавиши Shift на _дополнительной_ символьной раскладке, ⒊ С помощью значения $$symbol(см. @wikiarticle_2820). • {55FFFF symbol2} – дополнительная символьная раскладка. Открывается: ⒈ По нажатию клавиши Shift на символьной раскладке, ⒉ С помощью значения $$symbol2(см. @wikiarticle_2820). • {55FFFF edittext} – раскладка редактирования текста. Открывается: ⒈ По _долгому_ нажатию клавиши Shift, ⒉ По нажатию клавиши с [кодом]@wikiarticle_2822 -611, ⒊ По нажатию клавиши с кодом -619, ⒋ С помощью значения $$edittext(см. @wikiarticle_2820). • {55FFFF smile} – раскладка смайликов. Открывается: ⒈ По _долгому_ нажатию клавиши Enter, ⒉ По нажатию клавиши с кодом -610, ⒊ По нажатию клавиши с кодом -620, ⒋ С помощью значения $$smile(см. @wikiarticle_2820). • {55FFFF num} – числовая раскладка. Открывается: ⒈ Если атрибут type тега , который создаёт поле ввода текста, имеет значение number (). ⒉ По нажатию клавиши с кодом -612, ⒊ По нажатию клавиши с кодом -621, ⒋ С помощью значения $$num(см. @wikiarticle_2820). • {55FFFF calc} – раскладка калькулятора. Скрытый Возможный префикс только один – hide. В настройках клавиатуры вы никак не можете ни включить, ни отключить, ни поменять скрытые раскладки. Все скрытые раскладки, расположенные в папке jbak2Keyboard/keyboards, будут показаны при нажатии клавиши с [кодом]@wikiarticle_2822 -628, которую вы можете разместить на раскладке с любым префиксом, кроме calc. Также конкретные скрытые раскладки можно открывать с помощью [атрибутов android:keyboard и android:longKeyboard]@wikiarticle_2820. По значению атрибута android:typeKeyboard тега <Keyboard> Обычный Прокручивающийся Калькулятор android:typeKeyboard="calculator" Возможный префикс только один – calc.
