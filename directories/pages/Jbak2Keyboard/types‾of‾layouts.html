Языковые раскладки Последняя языковая раскладка открывается:
  1. При вызове (открытии) клавиатуры
  2. При нажатии клавиши с кодом -2 на любой раскладке, которая не является последней языковой раскладкой
Основной способ перехода между языковыми раскладками – это коды переключения языков.
Также можно открыть раскладку определённого языка с помощью значений $$ru, $$en и т. д. (см. атрибуты android:keyboard и android:longKeyboard). Открытая таким образом раскладка – временная: клавиша с кодом -2 на открытой таким образом раскладке будет возвращать к последней языковой раскладке. Неязыковые раскладки
Скрытые раскладки Конкретные скрытые раскладки можно открывать с помощью атрибутов android:keyboard и android:longKeyboard. Также существует код -628. 
• {55FFFF symbol} – символьная раскладка.
Открывается:
⒈ По нажатию клавиши с кодом -2 на языковой раскладке,
⒉ По нажатию клавиши Shift на _дополнительной_ символьной раскладке,
⒊ С помощью значения $$symbol(см. @wikiarticle_2820).
• {55FFFF symbol2} – дополнительная символьная раскладка.
Открывается:
⒈ По нажатию клавиши Shift на символьной раскладке,
⒉ С помощью значения $$symbol2(см. @wikiarticle_2820).
• {55FFFF edittext} – раскладка редактирования текста.
Открывается:
⒈ По _долгому_ нажатию клавиши Shift,
⒉ По нажатию клавиши с [кодом]@wikiarticle_2822 -611,
⒊ По нажатию клавиши с кодом -619,
⒋ С помощью значения $$edittext(см. @wikiarticle_2820).
• {55FFFF smile} – раскладка смайликов.
Открывается:
⒈ По _долгому_ нажатию клавиши Enter,
⒉ По нажатию клавиши с кодом -610,
⒊ По нажатию клавиши с кодом -620,
⒋ С помощью значения $$smile(см. @wikiarticle_2820).
• {55FFFF num} – числовая раскладка.
Открывается:
⒈ Если атрибут type тега , который создаёт поле ввода текста, имеет значение number ().
⒉ По нажатию клавиши с кодом -612,
⒊ По нажатию клавиши с кодом -621,
⒋ С помощью значения $$num(см. @wikiarticle_2820).
• {55FFFF calc} – раскладка калькулятора.
