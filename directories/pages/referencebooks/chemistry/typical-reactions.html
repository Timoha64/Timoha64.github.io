Типичные реакции
Кислот \(\text{H}^{+1}_\text{n}(\text{R})^{−\text{n}}_1\)
  1. Реакция нейтрализации:
    • Кислота + Основание / Амфотерный гидроксид → Растворимая соль + H2O (обмен)
    • Кислота + Бескислородное основание → \(\underset{(\text{NH}_4)^{+1}_\text{n}(\text{R})^{−\text{n}}_1}{\small{Соль}}\) (соединение)
  2. Кислота + Основный оксид / Амфотерный оксид[2] → Растворимая соль + H2O (обмен)
  3. Кислота, которая сильнее кислоты соли[1] + Соль → Новая кислота + Новая соль ↑/↓ (обмен)
  4. Кислота + Металл, который активнее H2 и не находится в ⅠA или ⅡA группе → Растворимая соль + H2↑ (замещение)
[1] \(\underset{Увеличение\ \text{α}}{\xleftarrow{\text{H}_2\text{SO}_4,\ \text{HCl},\ \text{HNO}_3\ |\ \text{H}_3\text{PO}_4\ |\ \text{H}_2\text{S},\ \text{H}_2\text{SO}_3,\ \text{H}_2\text{CO}_3}}\)

Нестабильные (неустойчивые) кислоты:
H2SO3 → SO2↑ + H2O
H2CO3 → CO2↑ + H2O
H2SiO3 → SiO2↓ + H2O (в реагентах не берём, т. к. это единственная нерастворимая кислота)
и др.

В цепочках реакций берём HCl 		Оснований \(\text{M}^{+\text{n}}_1(\text{OH})^{−1}_\text{n}\)
  1. Реакция нейтрализации:
    • Основание + Кислота / Амфотерный гидроксид → Растворимая соль + H2O (обмен)
    • Бескислородное основание + Кислота → \(\underset{(\text{NH}_4)^{+1}_\text{n}(\text{R})^{−\text{n}}_1}{\small{Соль}}\) (соединение)
  2. Щёлочь + Кислотный оксид / Амфотерный оксид[2] → Растворимая соль + H2O (обмен)
  3. Щёлочь + Растворимая средняя соль → Новое основание + Новая соль ↑/↓ (обмен)
  4. Нерастворимое основание / Амфотерный гидроксид t→ Оксид металла + H2O (разложение)
Гидрат аммиака (гидроксид аммония):
\(\text{NH}_4\text{OH}→\text{NH}_3{↑}+\text{H}_2\text{O ⇄ NH}_3\cdot\text{H}_2\text{O}\)
Единственное летучее основание

    • Амфотерный оксид + H2O
